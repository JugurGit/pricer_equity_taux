# Librairie de Pricing – Equity & Taux

Ce projet est une librairie de pricing en C++17 pour options actions et produits de taux :

- Options vanilles européennes (call/put) sous Black–Scholes
- Options digitales, asiatiques, barrières
- Caplet, cap, swap, swaption (modèle de Black sur les taux)
